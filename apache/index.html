Este container foi criado como solução ao desafio de projeto Criando um Container de uma Aplicação WEB da dio.

O projeto proposto apresenta a implementação de um servidor apache em um container. Além desta configuração, também foi integrado um container com uma API em .NET de uma lista de tarefas, que roda em uma sessão de banco de dados local.
